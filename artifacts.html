MonopolAR

A Monopoly-style AR game built with a team of 4 using Unity AR Foundation. Overlay a virtual game board on a physical one using image tracking. Runs on Android with a desktop version available on GitHub.

Skills: C#, AR Unity, basic 3D Modeling

Burglar Trap

A 2D tower defense game using LeapMotion for gesture controls. Defend against burglars with motion-tracked guns, grenades, and time manipulation via hand gestures.

Skills: C#, LeapMotion, Unity, Illustrator

A 2D shooter in space with pirate ships vs. Viking enemies. Features ship upgrades and boss battles. Developed for SkillsUSA, winning state gold and competing nationally.
